# Practick_WebApp_Ahmadeev
Простая игра с боковой прокруткой, в которой игрок управляет птичкой, летающей между трубами, не задевая их. Пролет между каждой группой труб приносит 1 очко, а столкновение с трубами или падение на землю заканчивают игру. Посмотрите, как долго вы сможете продержаться, и установите рекорды.
